 BIGS Overview

Broadband Information Gateway System เป็น Application ที่พยายามรวบรวมข้อมูลของโครงข่าย Broadband มีจุดประสงค์เพื่อ
1. ภาระหลักคือ Software ที่ทำหน้าที่เป็น Jigsaw ต่อเติมส่วนที่หายไปของโครงข่าย BroadBand
2. Focus ไปที่รายงานของแต่ละส่วนงาน ในโครงข่าย Broadband
3. อำนวยความสะดวกให้ผู้บริหารของโครงข่าย Broadband ในการดูข้อมูลสรุปแบบสั้นๆ
4. ไม่ทำหน้าที่ Software NMS ของระบบ Network
